# About

HeyCater Menu Server contains the backend code.

## Architecture

We use MVC code architecture.

# Get started

## Installation of gems

```bash
bundle
```

## Start up development server

```bash
cd bin

rails server
```

## Run test

```bash
bundle exec rspec
```

## Format file

```bash
rubocop -a file-path
```
